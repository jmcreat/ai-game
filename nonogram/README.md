# Nonogram Galaxy

우주/갤럭시 테마의 노노그램 퍼즐 게임입니다. Python + Pygame으로 제작되었으며 Windows / macOS / Linux에서 실행 가능합니다.

---

## 실행 방법

### Mac / Linux
```bash
cd nonogram
pip install -r requirements.txt
python main.py
```

### Windows
`run_windows.bat` 더블클릭 (자동으로 pygame 설치 후 실행)

---

## 게임 모드

| 모드 | 설명 |
|------|------|
| 스테이지 모드 | 손수 제작된 20개 퍼즐 (별, 로켓, 우주인, 행성 등) |
| 무한 모드 | 크기/난이도 선택 후 무한 랜덤 퍼즐 생성 |
| 일일 챌린지 | 오늘 날짜 시드로 매일 새로운 12×12 퍼즐 |

---

## 조작법

| 입력 | 동작 |
|------|------|
| 왼쪽 클릭 | 셀 채우기 / 취소 |
| 오른쪽 클릭 | X 표시 / 취소 |
| 드래그 | 연속 채우기 |
| 힌트 버튼 | 비어있는 정답 셀 하나 공개 |
| 리셋 버튼 | 현재 퍼즐 초기화 |
| ESC | 메뉴로 돌아가기 |

---

## 비주얼 이펙트

- 레이어드 별빛 배경 (시차 스크롤)
- 성운 그라디언트 배경
- 셀 채우기 시 골드 스파크 폭발
- 오답 시 빨간 흔들림 (Shake)
- 행/열 완성 시 골드 라인 클리어 이펙트
- 퍼즐 클리어 시 무지개 폭죽 애니메이션
- 네온 글로우 UI

## 파일 구조

```
nonogram/
├── main.py          # 메인 루프 + SceneManager
├── puzzle.py        # 퍼즐 로직 + 랜덤 생성
├── renderer.py      # 그리드/힌트/네온 UI 렌더링
├── particles.py     # 파티클 시스템 + 별빛 배경
├── scenes.py        # 씬 구현 (메뉴/선택/게임/무한)
├── data/
│   └── stages.py    # 손수 제작 스테이지 20개
├── requirements.txt
├── run_windows.bat
└── save.json        # 자동 생성 (클리어 기록)
```
